# Lumina Atelier E-Ticaret Projesi

Profesyonel görünümlü, mobil uyumlu, çok sayfalı bir e-ticaret başlangıç projesi.

## Özellikler
- Şık ana sayfa (hero, marka anlatımı, öne çıkan ürünler, kategoriler)
- Ürün detay sayfası
- Sepete ekleme ve sepet görüntüleme
- Kullanıcı girişi (demo, localStorage tabanlı)
- İletişim ve Hakkımızda sayfaları
- Yönetici paneline uygun dosya yapısı
- Örnek ürün datası ile hazır başlangıç

## Çalıştırma
1. Proje klasörüne girin:
   ```bash
   cd -eticaret-sitesi
   ```
2. Basit bir yerel sunucu başlatın:
   ```bash
   python3 -m http.server 5500
   ```
3. Tarayıcıdan açın:
   ```
   http://localhost:5500
   ```
