# Admin Paneli için Hazır Yapı

Bu klasör, yönetici paneli geliştirmesi için ayrılmıştır.

Önerilen genişletmeler:
- `admin/index.html`: Yönetim dashboard ekranı
- `admin/products/`: Ürün CRUD arayüzü
- `admin/orders/`: Sipariş yönetimi
- `admin/users/`: Kullanıcı yönetimi
- `admin/api/`: Backend endpoint entegrasyonu

Frontend ürün verileri şu an `assets/js/products.js` dosyasında tutulmaktadır.
Üretimde bu veri kaynağı API ile değiştirilmelidir.
